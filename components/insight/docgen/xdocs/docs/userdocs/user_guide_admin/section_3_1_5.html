To add a user to a user group:

  • Select the user in the user list.
  • Select the group you wish the user to belong to in the group list in the bottom left corner of the Users tab (Figure 3.1-2). If the group does not appear there but in the bottom right of the Users tab, the user already belongs to the group.
  • Once the user and group have been selected, click on the add user to group button.
The group should move from the group list in the bottom left to the list in the bottom right. The user is now a member of this group.
