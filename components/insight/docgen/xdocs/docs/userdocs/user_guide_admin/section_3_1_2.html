To delete a user from the system:

  • Select the user in the user list.
  • Click the remove user button.
  • You will be asked to confirm you wish to delete this user. If you select yes, this user will be deleted from the system. A user may fail to be deleted from the system if they own other data on the server (e.g. images or annotations). If this is the case OMERO.admin will warn you that this has happened.
