Depending on your operating system, OMERO.admin will either come as a simple zipped archive file, or as a bundled installation package. If you have downloaded the archive file, you will need to extract it with a program capable of opening .zip files. If you have downloaded the install package, double-click it to start the installation process and then follow the steps presented.

Once you have installed the software, go to Section 3 to see how to administer an OMERO server.
