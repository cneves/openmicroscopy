To create a new group:

  • Click on the create new group button of the top left of the Groups tab.

  • This will create a new group in the user list called NewGroup, you can now change the groupname of this group from NewGroup to the group you wish to create and fill the details of the group (see Figure 3.2.1-1).

    We can then save these new details by clicking the Save button. We have to click Save to update the NewGroup to the new details, including changing the name of the group.

NB
This new group will be created without having any members. To add members to this group, see Section 3.2.2.
