To add a text label ROI shape:

  1. Select the Text toolbar button
  2. Click on the area of the image in which the label is to be added.
  3. A text box with the word 'text' will appear.
  4. Double-click on the text box to add the desired label text.
  5. The label is now shown - resizing the area of the text label will adjust the size of the text.
Note that the text label will exist as an ROI with a single shape, and appear as such in the ROI Manager, Inspector and Results views. Double-clicking on the text ROI in the ROI Manager and Inspector allows you to edit the text.
