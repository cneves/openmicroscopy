Once you see thumbnails in the Hierarchy Viewer, you can view an individual image in more detail in the Image Viewer through the following options:

  1. Double-click on the image's thumbnail.
  2. Select the image thumbnail and then select View from the Edit menu.
  3. Right-click on the image thumbnail to display the Edit menu as for option 2.
