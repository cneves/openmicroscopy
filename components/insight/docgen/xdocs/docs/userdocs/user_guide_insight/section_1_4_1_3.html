The Image Viewer provides a number of additional tools:

  • Adjust the image properties using the Rendering Tool (see Section 3.9.3.5).
  • Magnify an image or area of interest within the image (see Section 3.9.3).
  • View in greyscale; or view additional information about the image (see Section 3.9.2).
  • Configure the optional scale bar (see Section 3.9.3.3).
