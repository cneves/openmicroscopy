A summary information for all ROIs of an image can be revealed by clicking on the Manager tab.

All ROIs and constituent ROI shapes are listed (see Figure 3.10.5.1-1).

For each ROI/ROI shape, the following essential information is shown:
  • For ROIs, all constituent ROI shapes (this will be the case where an ROI shape has been propagated across z and/or t to create a multi-dimensional ROI).
  • The unique ID number of the ROI/ROI shape.
  • For ROIs, the timepoints and/or z-sections across which it exists.
  • The type of ROI shape (one of Rectangle, Ellipse, Point, Line, Connector, Polygon or Text) is indicated by the appropriate symbol.
  • Annotation text.
  • Whether or not the ROI/ROI shape is currently visible.
