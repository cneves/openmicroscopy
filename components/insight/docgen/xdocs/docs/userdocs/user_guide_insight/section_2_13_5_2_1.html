Separate to text ROIs, an existing ROI shape can be annotated in one of three ways:

  1. Double-click on the shape; you can either edit the current annotation text or add new ones;
  2. Use the ROI Manager:
    1. Open the ROI Manager.
    2. Select the shape to be annotated: the selected ROI shape will be highlighted.
    3. Type the annotation; the annotation will appear as you type in the Annotation column.
    4. The annotation will appear within the ROI shape.
  3. Use the ROI Inspector:
    1. Select the shape to be annotated - the selected ROI shape will be highlighted.
    2. Open the ROI Inspector.
    3. Double click on the Annotation field.
    4. Edit the existing text, or add new texts.
See details of adjustments that can be made to the appearance of the annotation text in Section 3.10.5.4.1
