The Login page gives the access to the application. In order to connect to an OMERO server, the user has to specify:

  • the name of the server: field Hostname;
  • the port used: field Port;
  • a username: field Login;
  • a password: field Password.
for more details about fields see Section 3 or Section 2 .

 
Figure 4.1-1: Login Form

NB
Usernames are case sensitive, so alexandra would be a different user from Alexandra.

NB
If user has forgotten your password but know his username, and if he have a current email address in his OMERO registration details Forgotten Password? will be able to send him an email containing NEW password and informations..
