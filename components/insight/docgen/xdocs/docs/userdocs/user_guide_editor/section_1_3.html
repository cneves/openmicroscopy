In order to give new users an example of how protocols can be documented using OMERO.editor, some example files are accessible on-line. These can be accessed via the Help Menu > Open Demo File from URL, or by clicking the www-file icon on the toolbar.

Warning
You need to be on-line in order to access these demo files.

You will be presented with a dialog box that lists several available files. Choose the URL of a file by clicking on the link. Then click "Import" to download the file and open it in OMERO.editor.

 
Figure 2.3-1The import dialog, for choosing on-line example files to download.

You can now edit this file and use "Save-As" to save it in the location of your choice.
