New fields can be added to an OMERO.editor file by clicking the "New Field" button in the vertical tool bar to the right, or by choosing Edit-Template > Add Field, from the menu bar.

In order to edit the details of a field, click the field in the main window so that it becomes highlighted. Newly created fields will automatically be highlighted.

Once a field is highlighted, the details can be edited in the panel to the right. These details include:

  • The name of the field.
  • The "Field Type", which is the type of data (or experimental variables that you wish to store in this field. Eg. Number, Text, Date etc. The default value is "Fixed Field", which means that no experimental variables are stored in this field.
  • The description is an optional field for adding a description to the field.
  • An option url field can also be filled in.

Additional details exist for different types of field. For example, a Number field allows users to specify a default value and units.
