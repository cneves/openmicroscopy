Welcome to OMERO.editor

This document provides a guide to the Beta version of the OMERO.editor. This stand-alone application is designed to make it easier for scientists to record their experimental metadata.

The software was released for general use in June 2008.

This guide includes the following sections:

  • Section 2 - Getting started with OMERO.editor. Installing the software, and a brief guide to creating and editing documents.
Enjoy using the software!
