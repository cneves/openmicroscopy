Installing OMERO.Blitz Server

This document gives details on how to install the OMERO.Blitz Server and the software it is dependant on.

Prerequisite Software

The following software is required for the installation of OMERO.Blitz:

  1. IcePy 3.2.1 or greater
  2. Macports or a version of Readlink with the -f option
  3. Twisted Python 2.5.0
  4. zope.interface 3.3.0
