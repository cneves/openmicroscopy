Troubleshooting The OMERO.Blitz Server Install

To Rebuild the Database

  1. dropdb omero3
  2. createdb omero3
  3. java omero reload-db or dist/setup-db

Find the Password in Database

psql omero3 -c "select * from password"

Ports Required by OMERO.Blitz

  1. 4444
  2. 1098
  3. 1099

To Check that the Ports are Open

netstat -nav | grep PORTNO
Example

netstat -nav | grep 1099

tcp 0 0 0.0.0.0:1099 0.0.0.0:* LISTEN
