Setting Configuration Files

Setting Up The Environment

Add the following locations to your $PYTHON_HOME path

  1. PYTHON_HOME=/Library/Frameworks/Python.framework/Versions/2.5/
  2. PYTHON_HOME=$PYTHON_HOME:/YOUR_OMERO_DIR/dist/lib

Configuring OMERO.Blitz files

To enable the blitz server the omero.properties file in OMERO_DIR/etc needs to be configured

  1. ice.home=YOUR_ICE_PATH
  2. no.ice=false
