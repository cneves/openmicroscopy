Building the OMERO.Blitz Server

Building the Database

java omero rebuild-db

Clean the build

java omero clean cleantools

Build OMERO.Blitz

NOMAKE=1 java omero quick build dist withtools

Running OMERO.Blitz

cd YOUR_OMERO_DIR/dist/blitz
The first time you want to run the server type chmod +x ./blitz
JAVA_OPTS=-Xmx256M ./blitz start
Warning
if macports not installed change BLITZEXE='readlink -f $0' in file YOUR_OMERO_DIR/dist/blitz to BLITZEXE='$0'

Testing your OMERO.Blitz Server

./setup.py test -s test.scripts.harness
in YOUR_OMERO_DIR/dist/etc/ice.config omero.user=root
