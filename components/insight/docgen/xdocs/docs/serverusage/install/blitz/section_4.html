Setting up the Weblitz Interface

The Weblitz Interface Dependancies

To actually use weblitz you'll need django and pysqlite, for which has a small script that grabs and compiles both for inplace usage (i.e. not system wide).

Just run python weblitz_prepare.sh

Currently weblitz has prerequisites for

  1. pysqlite-2.3.5
  2. django SVN revision 6652+

If you don't want, can't compile and install django and, or pysqlite this way, just grab them and install system wide by whatever means you have available. Weblitz currently targeting pysqlite-2.3.5 and django's trunk from svn at revision 6652, but newer versions will almost certainly work.

The Weblitz Interface in Detail

A few other dependencies need to be satisfied to run the weblitz app, but if you don't want to actually use weblitz but just see how it connects and talks to the blitz server, you can check weblitz/blitzcon/blitz_connector.py that has all the logic in.

Running the Server

When all is done, run python manage.py runserver

Visit localhost:8000/admin to set up at least one Stored Connection. The base path will be the url you need to use to access that particular connection.

Note
Default username/password for administrator is admin:omero
