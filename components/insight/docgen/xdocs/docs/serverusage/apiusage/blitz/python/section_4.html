The OMERO.Blitz API

Introduction

The OMERO.Blitz API is contructed using ICE's slice definition language. A specification language for ICE it defines the contract between client and server and persistant data. In the next sections we will introduce the slice definitions for the OMERO.Blitz server, and show how to use them with Python.

The OMERO.Blitz server functionality is grouped under a number of services. Each service may be accessed via a Session object which is created by the Client object. The services are listed below.

  1. OMERO.Blitz Stateless Services
    1. AdminService
    2. ConfigService
    3. PixelsService
    4. PojosService
    5. QueryService
    6. RepositoryInfoService
    7. TypesService
    8. UpdateService
  2. OMERO.Blitz Stateful Services.
    1. RawFileStore
    2. RawPixelsStore
    3. RenderingEngine
    4. ThumbnailStore
