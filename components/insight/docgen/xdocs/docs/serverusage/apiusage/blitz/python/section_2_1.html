OMERO.Blitz Simple Types





These are basic types that have been wrapped by the 
OMERO.Blitz API. Each type is proceeded by an R - so RInt, RLong, RDouble.



Each OMERO.Blitz type is derived from a base type called RType.

Creating an RType object


To create an RType is simple. string = omero.RString("A String")

Retrieving a value from an RType Object


To access the contents of an RType object, just access it's val member.


Example Code



id = omero.RLong(pixelsID)
idValue = id.val





Output of an RType from console



>>> import omero
>>> a = omero.RLong(100)
>>> a
object #0 (::omero::RLong)
{
    val = 100
}
>>> a.val
100
