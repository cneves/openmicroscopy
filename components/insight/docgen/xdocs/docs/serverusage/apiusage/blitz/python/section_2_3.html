Creating an RType ADT object


To create an more complex RType is simple. string = omero.RList()



Example Code


The following code snippet creates a list of 3 RType objects.



>>> import omero  
>>> list = omero.RList()
>>> list.val = [omero.RString("Donald"), omero.RInt(10), omero.RLong(100)]
>>> list
object #0 (::omero::RList)
{
    val = 
    {
        [0] = object #1 (::omero::RString)
        {
            val = Donald
        }
        [1] = object #2 (::omero::RInt)
        {
            val = 10
        }
        [2] = object #3 (::omero::RLong)
        {
            val = 100
        }
    }
}





Retrieving a value from an RType Object


To access the contents of an RType object, just access it's val member.


Example Code



>>>list.val
[object #0 (::omero::RString)
{
    val = Donald
}, object #0 (::omero::RInt)
{
    val = 10
}, object #0 (::omero::RLong)
{
    val = 100
}]
>>>list.val[0].val
'Donald'





Output of an RType from console



>>> import omero
>>> a = omero.RLong(100)
>>> a
object #0 (::omero::RLong)
{
    val = 100
}
>>> a.val
100





Creating an RMap Object


To create an RMap object is slightly different as we use a string for the key and not an omero.RString.



>>> import omero
>>> map = omero.RMap()
>>> map.val = {"Donald":omero.RInt(10)}
>>> map
object #0 (::omero::RMap)
{
    val = 
    {
        key = Donald
        value = object #1 (::omero::RInt)
        {
            val = 10
        }
    }
}
>>>map.val.get("Donald")
object #0 (::omero::RInt)
{
    val = 10
}
