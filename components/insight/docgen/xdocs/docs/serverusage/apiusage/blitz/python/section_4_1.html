Slice and OMERO.Blitz





Introduction



The OMERO.Blitz API is contructed using ICE's slice definition language. 
A specification language for ICE it defines the contract between client and 
server and persistant data.



The slice definitions for the OMERO.Blitz server are located at 
YOUR_OMERO_DIR/components/blitz/resources/omero. The slice files are broken it 
different sections. The omero/ ice files are the central definitions of the blitz
implementation. They provide an Ice facade over the existing
OMERO.server architecture. Due to the increased overhead of
creating implementations in multiple languages, the mappings are
simplified and work mostly as simple data transfer objects.

	API.ice which contains the slice definition for the services and session calls.
