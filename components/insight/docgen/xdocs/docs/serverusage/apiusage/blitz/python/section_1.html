Overview of the OMERO.Blitz API

This document gives details on how to develop code to connect to and interact with the OMERO.Blitz server.The Blitz server uses the Internet Communication Engine(ICE) to communicate between client and server.

Available Bindings are available for OMERO.Blitz

Currently OMERO.Blitz has bindings for

  1. C++
  2. Java
  3. Python

What is in this Document

  1. Explanation of the OMERO.Blitz specific Types
  2. Example code to create a connection to the server and use the various methods in the API.
  3. Discussion on the ICE, slice definition language.
  4. List of all the API calls availble in the OMERO.Blitz server.
