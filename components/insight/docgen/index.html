OMERO.insight Documentation Site

Under Construction

This site aims to provide a centralized place where documentation on OMERO can be found. We provide documents at different levels and to different audiences. The available documents can be grouped in two categories: technical and users documents. In the first category fall documents that describe the OMERO.insight software as a whole (like the Software Architecture Document and the Build and Deployment Document), more detailed design documents on each software component, API usage documents and tutorials, and of course detailed documentation for each class (JavaDoc). The second category encompasses all the information that is relevant to the end users, like help documents and tutorials.

Documents currently available:

  • Client tools documentation
  • JavaDoc (html | no-frames html). Programmers documentation.
  • Software Build and Deployment Document (html | no-frames html | pdf). Describes the build system and how to build the various projects artifacts. Relevant to system administrators that want to build and deploy OMERO.insight from scratch. Also an handy reference document to developers.
  • Asynchronous Data Views design document (html | no-frames html | pdf). Details usage and design of the upcoming API that can be used to access the data services asynchronously.
